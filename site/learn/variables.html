Variables

Variables are fundamental to programming in both Java and Python. They act as containers for storing data values, such as integers, decimals, or strings.

Java and Python have different syntaxes when creating variables. In Java, you need to declare a specific type depending on what you want the variable to store.

Java Variables

Here is an example showcasing how to define variables using different types in Java.

In Java, variables must be declared with a specific data type:

  • int for integers (whole numbers)
  • double for numbers with decimals
  • char for single characters
  • String for multiple characters (words or sentences)
  • boolean for true/false values
Java Example
// The syntax to create a variable in Java is [type] [name] = [value];
char letter = 'W';
String word = "Word";
double dec = 0.2;
int num = 2;
boolean bool = true;
Output

Variables can also be used in mathematical expressions:

Java Example
int x = 5;
int y = 3;
int sum = x + y;
System.out.println("The sum is: " + sum);
Output
The sum is: 8
Python Variables

In Python, variables do not need to be declared with a specific data type. The type is inferred at runtime:

Python Example
# The syntax to create a variable in Python is [name] = [value]
x = 5
y = 3
sum = x + y
print("The sum is:", sum)
Output

The sum is: 8

Python allows you to change the type of a variable by assigning a new value:

Example
x = 5
x = "five"
print("x is now:", x)
Output

x is now: five

There are a few other data types, but these are some of the most common ones. As you can see, there was no output because we are only defining variables, not calling them.

You can attach variables to print statements as well:

Java Example
String name = "Vikram";
System.out.println("Hello, " + name + "!");
Output

Hello, Vikram!

Python Example
name = "Vikram"
print("Hello, " + name + "!")
Output

Hello, Vikram!
