If Statements

If statements are used to execute a block of code only if a specified condition is true.

Java If Statements

Here is an example of an if statement in Java:

Java Example
public class Main {
    public static void main(String[] args) {
        int number = 10;
        if (number > 0) {
            System.out.println("The number is positive.");
        } else {
            System.out.println("The number is not positive.");
        }
    }
}
Output

The number is positive.

10
Java Else-If Statements

Here is an example of using if, else if, and else statements in Java:

Java Example
public class Main {
    public static void main(String[] args) {
        int number = 10;
        if (number > 0) {
            System.out.println("The number is positive.");
        } else if (number < 0) {
            System.out.println("The number is negative.");
        } else {
            System.out.println("The number is zero.");
        }
    }
}
Output

The number is positive.

10
Python If Statements

Here is an example of an if statement in Python:

Python Example
number = 10
if number > 0:
    print("The number is positive.")
else:
    print("The number is not positive.")
Output

The number is positive.

10
Python Else-If Statements

Python Example
