Loops

Loops are a great way to run a section of code more than once. There are two main types of loops, for loops and while loops.

Here is an example of a simple for loop:

Java Example
for (int i = 0; i < 1; i++) {
   System.out.println("Hello! " + i);
}
Output

Hello! 0

i < 1

As seen above, the variable i is a value that increments until a certain limit, which is when the loop stops.

Similar to for loops, while loops can also be used to run code multiple times.

Java Example
int i = 0;
while (i < 1) {
   System.out.println("Hello! " + i);
}
Output

Hello! 0

i < 1
