Functions

Functions are blocks of code that perform a specific task. They can take inputs, called parameters, and return outputs.

Java Functions

Here is an example of a function that adds two numbers in Java:

Java Example
public class Main {
    public static void main(String[] args) {
        int result = add(3, 5);
        System.out.println("The sum is: " + result);
    }

    public static int add(int a, int b) {
        return a + b;
    }
}
Output

The sum is: 8

3
5
Python Functions

Here is an example of a function that adds two numbers in Python:

Python Example
def add(a, b):
    return a + b

result = add(3, 5)
print("The sum is:", result)
Output

The sum is: 8

3 5

Functions can take multiple parameters, return different types of values, and perform complex operations. They help in organizing code and making it reusable.
