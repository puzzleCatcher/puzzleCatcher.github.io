Print Statements

The print() function is used to display output text. It is commonly used to print strings and variables, but is also very helpful for debugging.

Java Example
System.out.println("Hello, world!");
Output

Hello, world!

You can attach variables to print statements as well:

Example
name = "Vikram";
System.out.println("Hello, " + name + "!");
Output

Hello, Vikram!

In Java, there is a difference between System.out.print() and System.out.println(). Here is an example:

Example
System.out.println("Hello");
System.out.print("World, ");
System.out.print("this is Vikram!");
Output

Hello
World, this is Vikram!

System.out.println() prints a string and moves to a new line, whereas System.out.print() stays on the same line.

These are some basic examples of using print statements. To learn more, view the resources on the Learn page.
