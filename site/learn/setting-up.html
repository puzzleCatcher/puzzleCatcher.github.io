Setting Up

To set up your code in a Java environment, you first need to create a file to store your code in. Create a file and call it Main.java.

Once you have your file ready, Java needs a class and a method to run your code. Create a function that looks like this in your file:

Java Example
public class Main {
  public static void main(String[] args) {
        // Your code here
  }
}

Unlike Java, Python doesn't require any methods or classes to run code. You can start writing code immediately.
